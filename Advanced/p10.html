CSS PSUEDO-CLASSES..!

The CSS Psuedo Classes allow you to style the dynamic states of an elememt of an element such as hover,active and focus-state, as well as elements that are existing in the document tree but cant be targeted via use of other selectors without adding any IDs or classes to them. for Ex, targeting the first or last child elements.

A Psuedo-Class starts with a colon(:) Its syntax can be given with:

selector:psuedo-class { property: value; }
